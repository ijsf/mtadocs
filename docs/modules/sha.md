<pageclass class="#0099AA"></pageclass> SHA is a module implementing SHA hashing for lua.

Provided Functions
------------------

-   sha1
-   sha224
-   sha256 (You may want to use the MTA function [sha256](/docs/sha256.md "wikilink"))
-   sha384
-   sha512
