Sends a message to the channel or a nickname

Syntax
------

``` lua
bool ircSay(string to, string message)
```

### Required arguments

-   **to:** The channel name, or a nickname.
-   **message:** The message to send..

### Returns

Returns true, otherwise false when giving wrong arguments.
