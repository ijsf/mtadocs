Gets color of the gang.

Syntax
------

``` lua
int int int getGangColor ( string Gang )
```

### Required Arguments

-   **Gang:** ID of the gang that you want to get colors of

### Returns

-   **R:** Amount of red from 0 to 255
-   **G:** Amount of green from 0 to 255
-   **B:** Amount of blue from 0 to 255
